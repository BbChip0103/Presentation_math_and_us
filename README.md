1. 영상처리 - 그라데이션 (1.ImageProcessing)
영상처리라고 하기엔 좀 부끄럽지만, 삼각함수를 응용한 그라데이션 예제입니다.
'수학이 없으면.raw'파일은 반복문을 이용하여 0~255의 값을 한 줄씩 입력한 사진입니다.
'하지만 삼각함수를 쓴다면 어떨까1, 2.raw'파일은 삼각함수 Sin파형의 0~∮/2까지의 값을 이용하여, 255배율로로 반복문을 이용하여 넣어준 사진입니다.
단순히 삼각함수를 곱하는 것으로, 비슷하면서도 다른 효과를 낼 수 있는 것을 알 수 있습니다.

2. 나선형 출력하기 - Pretest_1_1 (2.Pretest_1_1(math))
극좌표에서는 원을 그리기 쉽다는 것이 포인트인 문제입니다.
극좌표는 거리와 각도만으로 위치를 나타내는 좌표계인데, 원을 그리기 편하다는 것이 장점입니다.
원을 그릴 때 사용하는 도구인 컴퍼스를 생각하면 쉬운데, 반지름만큼 거리를 정하고, 각도만 돌려주면, 원이 매우 쉽게 그려집니다.
여기서 착안하여, 각도는 계속 돌려주면서 거리만 조절해준다면 나선이 그려진다는 것을 알 수 있었습니다.
그리고 콘솔창은 직교좌표계를 쓰기 때문에, 극좌표를 직교좌표로 변환해주면 완성입니다.
*조금 더 정교하게 만들어보고자 좌표값을 반올림 하였습니다.
